# PHP Service

A PHP microservice for the Pantalasa platform built with Slim Framework.

## Requirements

- PHP 8.2+
- Composer 2.6+

## Installation

```bash
composer install
```

## Usage

```bash
php -S localhost:8080 -t public
```

## Testing

```bash
composer test
```

## Static Analysis

```bash
composer analyse
```

## Contributing

1. Create a feature branch
2. Make changes
3. Run tests and static analysis
4. Submit a PR
